# C# Learning Lab

Repositório criado para organizar exercícios, testes e pequenos projetos desenvolvidos durante a minha aprendizagem em programação.

## Conteúdos praticados

- C#
- Programação Orientada a Objetos
- Classes, objetos, métodos e propriedades
- Listas e enums
- SQL básico
- Git e GitHub
- Pequenos projetos de consola

## Objetivo

Este repositório tem como objetivo acompanhar a minha evolução na área de desenvolvimento de software e consolidar conceitos através da prática.
